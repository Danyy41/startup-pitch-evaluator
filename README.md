# startup-pitch-evaluator
Domain-specific LLM fine-tuned with QLoRA to evaluate startup pitches — outputs strengths, weaknesses, market opportunity, and risk score.
 Startup Pitch Evaluator

Startup Pitch Evaluator is a domain-specific LLM fine-tuned using QLoRA/PEFT on startup pitch evaluation data. The system analyzes startup descriptions and generates structured venture-capital style assessments including strengths, weaknesses, market opportunity, and a quantitative risk score. Built on Mistral-7B-Instruct with Transformers, PEFT, TRL, MLflow, and Hugging Face, the project demonstrates parameter-efficient fine-tuning, experiment tracking, model deployment, and applied AI for investment decision support.
