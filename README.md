# 🚀 Startup Pitch Evaluator

Startup Pitch Evaluator is a domain-specific LLM fine-tuned with QLoRA/PEFT to evaluate startup pitches like a venture capitalist — you give it a startup description and it returns the strengths, weaknesses, market opportunity, and a risk score from 0 to 10, making it useful for venture capital and entrepreneurship. It is built on the Mistral-7B-Instruct base model and fine-tuned using QLoRA (4-bit quantization plus LoRA adapters) so it trains on a single GPU, with experiments tracked in MLflow and the final model shared on the Hugging Face Hub. To use it, install the dependencies with `pip install -r requirements.txt`, train the model with `python src/train.py`, test it on a new pitch with `python src/inference.py`, and publish it with `python src/push_to_hub.py`. The project is built with transformers, peft, trl, bitsandbytes, mlflow, and huggingface_hub, and is currently a work in progress as the training dataset is expanded and hyperparameters are tuned.
